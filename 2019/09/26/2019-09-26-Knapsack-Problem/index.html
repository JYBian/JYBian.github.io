Knapsack Problem

I. Introduction

I.1 What is Knapsack Problem

Knapsack Problem decribes the kind of situation, where a group of objects has been given. Each of them has it’s own weight and value. We need to optimize our choice to gain the greatest total value within a limitation of weights.

I.2 Categories

Different categories: A (0-1 Knapsack Problem), B, C, D, E, F, G, H.

II. A (0-1 Knapsack Problem)

II.1 Description

Suppose we have N objects and a knapsack that can hold V kg staffs. The weight of the ith object is w[i] kg with val[i] as the value. Find out which object(s) to put in tomake the value maximum without excess the weight limitation.

II.2 Solution

We are going to appy Dynamic Programming here to solve the problem. The stage we need to consider is the number of objects in the package and the status is the capacity left in the Knapsack. Suppose f[i, v] stands for the maximum value that we can have wehn choosing from i objects, and v stands for the capacity of knapsack. Then the status transmission equation would be: $f[i][v] = max{f[i-1][v], f[i-1][v-w[i]]+c[i]} $.

II.3 Optimization for space

In the above equation, 2D-array is needed, but actually, we just need 1D-scrolling array for the recursion. Since we are using f[v] to store the value for each iteration, and what we need is to get f[i][v] using f[i-1][v] and f[i-1][v-w[i]], so whenwe are doing the recursion, we just need not to cover the value for f[v-w[i]] when calculating f[v], so we can apply reverse-order loop here, the pseudo-code can beshown below:

III. B Type

III.1 Description

Suppose we have N kinds of objects and a knapsack with V capacity, there is no limition on the number of each kind of object. The weight for the first object is w[i] and its value is val[i]. Try to find the way to arrange these pbjects in the knapsack so that we can achieve the highest value without excess the weight limitation.

III.2 Solution

The difference between A Type and B Type is that in B Type , we have infinite number for each object. We an change the equation in A Type a little bit, let f[i][v] stands for the maximum value when we place i kinds of objects in a knapsack with v limitation, and the equation goes like: $f[i][v] = max{f[i-1][v], f[i-1][v-kw[i]]+kc[i]|0 <= k*w[i] <= v}$. The time complexity would be O(n^3) and the space complexity would be O(n^2).
