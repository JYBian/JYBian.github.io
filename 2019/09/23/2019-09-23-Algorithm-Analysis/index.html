Mathematical Model

1. Appromixation

  • $\frac{N^3}{6}-\frac{N^2}{2}+\frac{N}{3}$ ~ $\frac{N^3}{6}$.
  • $~f(N)$ can be used to represent all the functions deveided by $f(N)$ that will approach 1 when the altitude of $N$ becomes bigger.

2. Order of Magnitude

  • The order of magnitude for $\frac{N^3}{6}-\frac{N^2}{2}+\frac{N}{3}$ is $O(N^3)$
  • The order of magnitude is separate from the realization of algorithm, i.e. the order of magnitude for one algorithm is $O(N^3)$ has nothing to do with whether it is realized using Java or using C++.

3. Inner Loop

  • The most frequently used command decides the total time for a program to run, such command is called inner loop of a program.

Things to Notice

1. Big Constant Number

  • When coping with approximation problems, if there is a big constant before a lower term, the approximation result may not be accurate.

2. Cache

  • Computer use cache to organize the memory, visiting a neighbored element in an array will be much faster than visiting non-neighbored one.

3. Randomize Algorithm

  • When testing algorithm, try to mess up the input to reduce the dependence of input for an algorithm.
